# Antigravity Control Horario

Proyecto de control de horario desarrollado durante el **Sprint 1** de Fundamentos Reales y Vibe Coding con Antigravity.

## Objetivos del Proyecto
- Entender el concepto de **Vibe Coding**.
- Diferenciar entre entornos locales y web para el desarrollo.
- Dominar el uso de herramientas de IA generativa para el código.
- Implementar lógica básica de negocio mediante JavaScript.

## Estructura del Repositorio
- `01_fundamentos.md`: Documento teórico sobre Vibe Coding y herramientas.
- `/ejercicios_js_basicos`: Micro-ejercicios de lógica en JavaScript.

## Reglas de Pair Programming
- **Día 1**: Driver (Antigravity) y Navigator (Usuario).
- Cambio de roles programado.
- Mínimo 3 commits por integrante/sesión.
